Welcome to the Web-Dev Challenge!

You think you have what it takes to make a beautiful, breathtaking website? Let's find out!

About

In this event, contestants will be provided a starting template along with a set of various requirements that contestants must adhere to while building their front-end website.

Contestants will be given about 2 days to submit their code.

The websites submitted will be judged based on several features like creativity, design, readability of the code, etc.

Rules

  • You will be provided a document containing information about the project to be built along with a starting
    template. You would be required to fulfill the requirements in the document and submit the project
  • The document would be provided to you one day prior to the judging phase
  • You are free to use CSS libraries like Bootstrap and TailwindCSS and JS libraries like aos.JS
  • Your code will be judged on the basis of readability, comments. Your website would be judged on the basis of
    creatvity, design, layout, typography, color scheme, responsiveness, animations (optional)
