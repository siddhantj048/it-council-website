About Us

This event is done virtually. This is a one day challenge, where the contestants are supposed to crack cryptic messages using their IT knowledge, where one challenge leads to another to find the final code. The prerequisites are basic knowledge regarding computers, some basic programming knowledge, and curiosity

Rules

  • You will be provided a document containing information about the project to be built along with a starting template. You would be required to fulfill the requirements in the document and submit the project.
  • The document would be provided to you one day prior to the judging phase.
  • You are free to use CSS libraries like Bootstrap and TailwindCSS and JS libraries like aos.JS
  • Your code will be judged on the basis of readability, comments. Your website would be judged on the basis of creatvity, design, layout, typography, color scheme, responsiveness, animations (optional)
